# Add / Drop Process – Frequently Asked Questions

Click a question to view the answer.<br>
Use Ctrl + F (Windows) or ⌘ + F (Mac) to search within this page.

---

<details>
<summary><strong>When does the Add / Drop period begin?</strong></summary>

The Add / Drop period begins at the start of registration.<br><br>

The period ends as follows:<br><br>

- Semester 1 (SEM 1): End of the second week of classes<br>
- Semester 2 (SEM 2): End of the second week of classes<br>
- Semester 3 (Summer): End of the first week of classes<br>

</details>

---

<details>
<summary><strong>What happens if I drop a module during the Add / Drop period?</strong></summary>

If you drop a module during the Add / Drop period, the following will occur:<br><br>

- The module will be removed from your academic record<br>
- The charges for the module will be removed from your student account<br>

</details>

---

<details>
<summary><strong>What happens if I add a module during the Add / Drop period?</strong></summary>

If you add a module during the Add / Drop period:<br><br>

- The cost of the module will be charged to your account<br>
- A waiver from the Registrar’s Office is required to avoid additional charges<br>

</details>

---

<details>
<summary><strong>Can I add or drop modules after the Add / Drop period ends?</strong></summary>

No.<br><br>

Adding or dropping modules is not permitted after the Add / Drop period has ended.

</details>

---

<details>
<summary><strong>Should I speak with anyone before adding or dropping a module?</strong></summary>

Yes.<br><br>

Students are strongly advised to consult their Academic Advisor before adding or dropping any module to ensure that it aligns with their programme requirements.

</details>

---

<details>
<summary><strong>What is required to qualify for the Add / Drop process?</strong></summary>

To access the Add / Drop platform, you must be a currently registered student.

</details>

---

<details>
<summary><strong>What is the procedure to add or drop a module?</strong></summary>

To add or drop a module, follow these steps:<br><br>

1. Visit the University’s website<br>
2. Select the <strong>Online Add / Drop</strong> option<br>
3. Enter the required information on the platform<br>
4. Use the Help option on the Add / Drop page if you need guidance<br>

</details>
