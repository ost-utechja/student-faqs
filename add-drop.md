# Add / Drop Process – Frequently Asked Questions

Click a question to view the answer.  
Use Ctrl + F (Windows) or ⌘ + F (Mac) to search within this page.

---

<details>
<summary><strong>When does the Add / Drop period begin?</strong></summary>

The Add / Drop period begins at the start of registration and ends as follows:

- Semester 1 (SEM 1): End of the second week of classes  
- Semester 2 (SEM 2): End of the second week of classes  
- Semester 3 (Summer): End of the first week of classes  

</details>

---

<details>
<summary><strong>What happens if I drop a module during the Add / Drop period?</strong></summary>

If a module is dropped during the Add / Drop period:
- The module is removed from your academic record  
- The charges for the module are removed from your account  

</details>

---

<details>
<summary><strong>What happens if I add a module during the Add / Drop period?</strong></summary>

If a module is added during the Add / Drop period:
- The cost of the module will be charged to your account  
- A waiver from the Registrar’s Office is required to avoid charges  

</details>

---

<details>
<summary><strong>Can I add or drop modules after the Add / Drop period ends?</strong></summary>

No. Adding or dropping modules is not permitted after the Add / Drop period ends.

</details>

---

<details>
<summary><strong>Should I consult anyone before adding or dropping a module?</strong></summary>

Yes. Students are strongly advised to consult their Academic Advisor before adding or dropping any module.

</details>

---

<details>
<summary><strong>What is required to qualify for the Add / Drop process?</strong></summary>

You must be a currently registered student in order to access the Add / Drop platform.

</details>

---

<details>
<summary><strong>What is the procedure to add or drop a module?</strong></summary>

To add or drop a module:

1. Visit the University’s website  
2. Select the <strong>Online Add / Drop</strong> option  
3. Enter the required information  
4. Use the Help option on the platform for guidance if needed  

</details>
